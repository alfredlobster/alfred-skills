# Escalation to Next Skills

## Purpose

Choose the next skill based on the main bottleneck after intake.

## Recommended next skills

### hypothesis-generator

Use when the problem is framed well enough and the next need is to generate candidate explanatory or intervention hypotheses.

Choose this when:
- the research goal is clear
- some evidence exists
- the main gap is idea generation or hypothesis expansion

### experiment-designer

Use when one or more hypotheses already exist and the next need is discriminating tests, controls, measurements, and feasibility planning.

Choose this when:
- the user already has hypotheses
- the brief contains strong candidate directions
- the main gap is execution design

### critic-reviewer

Use when the likely failure mode is weak reasoning, poor novelty, confounding factors, or overconfidence.

Choose this when:
- the brief contains strong assumptions
- the evidence base is thin or conflicted
- the user wants a challenge step before ideation or experiments

### ranking-and-decision

Use when multiple options already exist and the main need is prioritization.

Choose this when:
- there are several credible hypotheses or study directions
- constraints and decision criteria matter more than generation

### study-plan-generator

Use when the user needs an actionable near-term program of work.

Choose this when:
- the brief is mature enough to sequence activities
- there is enough clarity on objectives, constraints, and first experiments

## Default rule

If the brief is coherent but underdeveloped, recommend `hypothesis-generator`.
If the brief is coherent and already hypothesis-rich, recommend `experiment-designer`.
If the brief feels fragile or overconfident, recommend `critic-reviewer`.
