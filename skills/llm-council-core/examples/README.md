# LLM Council Example Runs

This directory contains example artifacts for the LLM Council workflow.

## Current example
- `internal-developer-platform/`

## How to think about the flow
1. Start with a hard question or design choice
2. Normalize it with `llm-council-intake`
3. Generate independent first opinions
4. Run anonymous peer review and rankings
5. Aggregate the standings
6. Ask the chairman to synthesize the final answer
7. Generate a final report
