# LLM Council Reporter

Create the final report for a council run.

## Purpose
Turn the full council run into a readable artifact that preserves process transparency.

## Required sections
1. Task / prompt
2. Council members
3. First opinions summary
4. Anonymous peer-review summary
5. Aggregate ranking table
6. Chairman final answer
7. Notes / uncertainty

## Rules
- Preserve the distinction between first opinions and chairman synthesis
- Do not hide disagreement if it exists
- Make aggregate rankings visible
- Keep the report auditable and easy to review
